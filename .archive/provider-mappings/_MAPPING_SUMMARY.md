## How to Use

1. Open each mapping file
2. For each OLD provider, review the top 5 suggested matches
3. Compare addresses (same address = same facility)
4. Mark your decision in the checkboxes
5. Return the marked files for script generation

---

## Decision Guide

**[REPLACE]** - Use when:
- Same address as new provider
- New provider has more complete information
- Old name is abbreviated version of new name

**[KEEP OLD]** - Use when:
- No good match in new roster
- Old provider is independent (not in this health system)
- Addresses don't match any new providers

**[DELETE]** - Use when:
- Old provider is duplicate of another old provider
- Misspelled or data entry error
