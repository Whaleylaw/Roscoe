# Wage Loss Documentation Checklist (Gap #13)

## Overview
Collect and verify documentation for lost wage claims.

---

## When Wage Loss Applies

### Triggers
1. Client reports missing work due to injuries
2. Doctor's off-work note in records
3. Client check-in flags `NOT_WORKING`
4. Demand preparation reveals lost time

---

## Required Documentation

### From Client
- [ ] Pay stubs (8-12 weeks before accident)
- [ ] Pay stubs (period of missed work)
- [ ] Tax returns (last 2 years if self-employed)
- [ ] Employment contract (if relevant)

### From Employer
- [ ] Employer verification letter
- [ ] Statement of wages/hours
- [ ] Dates of missed work
- [ ] Return to work date

### From Doctor
- [ ] Off-work note(s) with specific dates
- [ ] Work restrictions (if partial disability)
- [ ] Return to work clearance

---

---

## Calculation Methods

### Hourly Employee
```
Lost Wages = Hourly Rate × Average Hours/Week × Weeks Missed
```

### Salaried Employee
```
Lost Wages = Annual Salary ÷ 52 × Weeks Missed
```

### Self-Employed
- Use tax returns to establish average income
- May need accountant verification
- More documentation required

---

## Documentation Checklist by Type

### W-2 Employee (Standard)
- [ ] Last 3 pay stubs before accident
- [ ] Pay stubs during missed work period
- [ ] Employer verification letter

### Self-Employed
- [ ] Last 2 years tax returns (Schedule C)
- [ ] Bank statements showing income
- [ ] Client affidavit of lost income
- [ ] Accountant letter (if available)

### Gig/1099 Worker
- [ ] 1099 forms from prior year
- [ ] Payment records/invoices
- [ ] Platform earnings reports

---

## Common Issues

### Issue: No Pay Stubs
- Request W-2 from employer
- Request wage history from employer
- Use tax returns as backup

### Issue: Partial Disability
- Document reduced hours
- Calculate difference in earnings
- Get doctor's work restrictions

### Issue: Self-Employed
- More scrutiny from insurance
- Need strong documentation
- Consider expert if large claim

---

## For Demand Package

Include in demand:
1. Off-work notes from doctor
2. Employer verification letter
3. Pay stubs/wage records
4. Calculation of total lost wages
5. Summary table of missed work dates
