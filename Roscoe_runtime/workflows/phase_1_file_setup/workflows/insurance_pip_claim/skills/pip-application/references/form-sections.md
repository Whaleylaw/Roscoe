# KACP Application Form Sections

## Form Overview

The KACP (Kentucky Assigned Claims Plan) Application is the universal PIP application form accepted by all Kentucky auto insurers. It contains 7 main sections.

## Section 1: Personal Information

**Purpose**: Identify the applicant

**Fields**:
- Full legal name (must match ID)
- Contact phone numbers (home and work)
- Complete mailing address
- Date of birth
- Social Security Number

**Notes**:
- Name should be exactly as it appears on government ID
- SSN is required for claim processing
- Address is where correspondence will be sent

## Section 2: Accident Information

**Purpose**: Document the accident details

**Fields**:
- Date and time of accident
- Location/place of accident
- Brief description of what happened

**Notes**:
- Date should match police report
- Location should be specific (intersection, mile marker, etc.)
- Description should be factual, brief (2-3 sentences)

## Section 3: Vehicle/Insurance Information

**Purpose**: Identify vehicle and insurance status

**Fields**:
- Does applicant own a motor vehicle? (Yes/No)
- Insurance company name
- Policy number (if known)

**Notes**:
- This connects to PIP waterfall determination
- If coverage through vehicle occupant, list that insurer
- Policy number may not be known at time of application

## Section 4: Injury Information

**Purpose**: Describe injuries from the accident

**Fields**:
- Description of injuries
- Body parts affected
- Current symptoms

**Notes**:
- Be specific about body parts
- Include both immediate and ongoing symptoms
- Avoid exaggeration or speculation

## Section 5: Medical Treatment

**Purpose**: Document treating providers

**Fields**:
- Treating physician name and address
- Hospital name and address (if applicable)
- Other medical providers

**Notes**:
- List primary treating provider first
- Include EMS/ambulance if transported
- Hospital address is where treatment received

## Section 6: Employment Information

**Purpose**: For lost wage claims (if applicable)

**Fields**:
- Employer name and address
- Occupation
- Employment status

**Notes**:
- Required if claiming lost wages
- Can leave blank if not employed
- Include even if not initially claiming wages

## Section 7: Signature

**Purpose**: Certify accuracy and authorize release

**Fields**:
- Applicant signature
- Date of signature

**Notes**:
- Must be signed by applicant (or legal guardian)
- Signature certifies all information is true
- Date should be date of actual signing

## Completion Checklist

Before submitting, verify:

- [ ] All personal information accurate
- [ ] Accident date matches other documents
- [ ] Insurance information from waterfall determination
- [ ] Injuries described completely
- [ ] Medical providers listed
- [ ] Form signed and dated
