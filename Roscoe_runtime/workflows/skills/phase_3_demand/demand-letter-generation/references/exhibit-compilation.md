# Exhibit Compilation Guide

## Standard Exhibit Organization

### Exhibit Index

| Exhibit | Description | Required |
|---------|-------------|----------|
| A | Medical Records Index | Yes |
| B | Medical Records (all providers) | Yes |
| C | Medical Bills (itemized) | Yes |
| D | Medical Chronology | Yes |
| E | Accident/Police Report | If MVA |
| F | Photos (injuries, vehicle, scene) | If available |
| G | Wage Loss Documentation | If applicable |
| H | Property Damage Documentation | If applicable |
| I | Expert Reports | If any |

---

## Exhibit A: Medical Records Index

**Purpose**: Quick reference for adjuster to locate records

**Format**:
```
MEDICAL RECORDS INDEX

Provider                  | Type        | Pages | Exhibit
--------------------------|-------------|-------|--------
Norton Hospital ER        | ER Records  | 1-15  | B-1
Dr. John Smith, Ortho     | Office      | 16-45 | B-2
ABC Physical Therapy      | PT Records  | 46-82 | B-3
Louisville Imaging        | MRI Report  | 83-88 | B-4
```

---

## Exhibit B: Medical Records

**Organization**: 
1. By provider, chronologically
2. Tab each provider separately
3. Bates number if possible

**Order**:
1. Emergency/Initial treatment
2. Primary care
3. Specialists (ortho, neuro, pain)
4. Physical therapy/chiropractic
5. Imaging/diagnostics

**Include**:
- All office notes
- Imaging reports
- Procedure notes
- Referral letters
- Discharge summaries

**Exclude**:
- Billing-only documents (go in Exhibit C)
- Duplicate records
- Unrelated treatment

---

## Exhibit C: Medical Bills

**Format**: Summary table + itemized bills

**Summary Table**:
```
MEDICAL EXPENSES SUMMARY

Provider                  | Service Dates      | Amount
--------------------------|-------------------|----------
Norton Hospital ER        | 04/26/2024        | $3,500.00
Dr. John Smith, Ortho     | 04/28-08/15/2024  | $2,800.00
ABC Physical Therapy      | 05/01-07/30/2024  | $4,200.00
Louisville Imaging        | 05/05/2024        | $1,800.00
Pain Management Assoc.    | 06/01-07/15/2024  | $8,500.00
                                      TOTAL:  | $20,800.00
```

**Itemized Bills Should Include**:
- Date of service
- CPT codes
- ICD-10 codes
- Line item charges
- Provider name

---

## Exhibit D: Medical Chronology

**Include**:
- Full chronology document (from medical_chronology workflow)
- Page references to records
- Cited medical definitions

**Format**: Professional PDF with:
- Header with client info
- Chronological entries
- Comments column with definitions
- Source page references

---

## Exhibit E: Accident Report

**If Police Report Available**:
- Include full report
- Highlight key findings:
  - Fault determination
  - Witness statements
  - Citations issued
  - Diagram

**If No Police Report**:
- Note in exhibit index: "No police report - not reported"
- Include any available documentation (insurance photos, etc.)

---

## Exhibit F: Photographs

**Organize By**:
1. Vehicle damage (exterior, interior)
2. Injury photos (if any)
3. Scene photos
4. Property damage

**For Each Photo**:
- Label with date taken
- Brief description
- Source (client, police, insurance)

---

## Exhibit G: Wage Loss

**Include**:
- Off-work notes from physicians
- Pay stubs (3-6 months pre-accident)
- W-2s or tax returns (income verification)
- Employer verification letter
- Calculation worksheet

**Calculation Format**:
```
LOST WAGE CALCULATION

Pre-Accident Income:
  Hourly rate: $25.00
  Hours per week: 40
  Weekly income: $1,000.00

Days Missed: 45 days (9 weeks)
Lost Wages: 9 Ã— $1,000.00 = $9,000.00
```

## Exhibit H: Property Damage

**Include**:
- Repair estimates
- Photos of damage
- Total loss valuation (if totaled)
- Rental receipts
- Diminished value appraisal (if any)

## File Size Considerations

### If Package Is Too Large for Email

| Option | When to Use |
|--------|-------------|
| Secure file share | Most adjusters accept |
| CD/DVD | Traditional option |
| Split sends | Records separate from letter |
| Summary + full | Summary in email, full by mail |

### Size Reduction Tips
- Compress PDFs
- Use black & white for records
- Summarize rather than include duplicates
- Offer full records on request

---

## Quality Checklist

Before compiling final package:
- [ ] All exhibits present and accounted for
- [ ] Exhibit index matches actual contents
- [ ] Page numbers are accurate
- [ ] Records are legible
- [ ] Bills are itemized (not just statements)
- [ ] Photos are clear and labeled
- [ ] Chronology is finalized
- [ ] Package is organized professionally
- [ ] File size is manageable for sending
