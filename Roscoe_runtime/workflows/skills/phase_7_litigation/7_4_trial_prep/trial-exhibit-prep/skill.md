# Trial Exhibit Prep Skill

## Overview

Organize and prepare all exhibits for trial presentation.

## When to Use

Use when:
- Preparing exhibit list for court
- Organizing documents for trial
- Creating exhibit notebooks

DO NOT use if:
- Preparing deposition exhibits (simpler process)
- Managing discovery productions

## Workflow

### Step 1: Identify All Exhibits

Categories to consider:
- Medical records and bills
- Photos (accident, injuries)
- Documents (police report, etc.)
- Demonstratives
- Expert materials

**See:** `references/exhibit-list-format.md` for standard format.

### Step 2: Organize and Number

- Bates stamp if not already
- Create exhibit index
- Pre-mark for trial

### Step 3: Prepare Exhibit List

Per court requirements:
- Description
- Number/Bates range
- Objection status
- Foundation witness

**See:** `references/witness-list.md` for coordinating.

### Step 4: Create Trial Notebooks

- Witness folders
- Exhibit copies
- Deposition excerpts

**See:** `references/jury-instructions.md` for instruction prep.

## Output Format

```markdown
## Exhibit List: [Case Name]

| # | Description | Bates | Witness | Objection |
|---|-------------|-------|---------|-----------|
| 1 | [Item] | [#] | [Name] | [Status] |

**Total Exhibits:** [X]
**Stipulated:** [X]
**Contested:** [X]
```
