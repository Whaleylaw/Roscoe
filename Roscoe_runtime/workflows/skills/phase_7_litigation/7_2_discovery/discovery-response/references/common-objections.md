# Common Discovery Objections Reference

## Standard Objection Language

### Vague and Ambiguous

```
Plaintiff objects to this request as vague and ambiguous in that 
the term "[TERM]" is susceptible to multiple interpretations and 
does not reasonably apprise Plaintiff of the information sought. 
Subject to and without waiving this objection, Plaintiff responds 
as follows: [RESPONSE]
```

### Overly Broad

```
Plaintiff objects to this request as overly broad in scope in that 
it is not limited in time, subject matter, or relevance to the 
claims and defenses in this action. Subject to and without waiving 
this objection, Plaintiff responds as follows: [RESPONSE]
```

### Unduly Burdensome

```
Plaintiff objects to this request as unduly burdensome in that 
the time, expense, and effort required to respond is disproportionate 
to any possible benefit. Subject to and without waiving this 
objection, Plaintiff responds as follows: [RESPONSE]
```

### Not Reasonably Calculated

```
Plaintiff objects to this request as it seeks information that is 
not relevant to any claim or defense and is not reasonably 
calculated to lead to the discovery of admissible evidence. 
Subject to and without waiving this objection, Plaintiff responds 
as follows: [RESPONSE]
```

### Attorney-Client Privilege

```
Plaintiff objects to this request to the extent it seeks information 
protected by the attorney-client privilege. Documents withheld on 
this basis are identified in the privilege log served herewith.
```

### Work Product

```
Plaintiff objects to this request to the extent it seeks documents 
or information protected by the work product doctrine. Documents 
withheld on this basis are identified in the privilege log served 
herewith.
```

### HIPAA / Medical Privacy

```
Plaintiff objects to this request to the extent it seeks protected 
health information without proper authorization. Subject to and 
without waiving this objection, Plaintiff will produce medical 
records for treatment related to this accident upon receipt of 
appropriate authorizations.
```

### Compound

```
Plaintiff objects to this interrogatory as compound in that it 
contains multiple discrete questions within a single numbered 
interrogatory. Subject to and without waiving this objection, 
Plaintiff responds to the extent the question can be reasonably 
understood: [RESPONSE]
```

### Assumes Facts

```
Plaintiff objects to this request as it assumes facts not in 
evidence, specifically [ASSUMED FACT]. Subject to and without 
waiving this objection, Plaintiff responds: [RESPONSE]
```

---

## Responding Despite Objection

Always include substantive response where possible:

```
Subject to and without waiving the foregoing objections, Plaintiff 
responds as follows:

[Provide the responsive information you can provide]
```

---

## When to Object vs. Answer

| Situation | Recommendation |
|-----------|----------------|
| Request clearly improper | Object and don't answer |
| Request borderline | Object but answer anyway |
| Privacy concerns | Object, provide limited response |
| Privileged documents | Object, provide privilege log |

---

## Objections to Avoid

These objections are disfavored and may invite sanctions:

- "General objections" incorporated by reference
- Objections without good faith basis
- Objections that don't explain the problem
- Objecting to every request identically

---

## Preserving Objections

To preserve objections:
- State them in timely response
- Be specific about grounds
- Don't waive by providing full response without objection
