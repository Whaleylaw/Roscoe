# Post-Deposition Analysis Guide

## Phase 1: Transcript Review

### Step 1.1: Initial Review

**When Transcript Arrives:**
- Review within designated time for corrections (usually 30 days)
- Check for typographical errors
- Look for substantive irregularities
- Identify discrepancies from what was actually said

**Review Checklist:**
- [ ] Transcript received and logged
- [ ] Page count noted
- [ ] Names spelled correctly throughout
- [ ] Technical terms transcribed correctly
- [ ] No obvious gaps or "inaudible" notations that can be clarified

---

### Step 1.2: Errata Sheet Preparation

**When to Submit Corrections:**
- Typographical errors that change meaning
- Misheard words that affect substance
- Technical terms incorrectly transcribed

**Errata Sheet Format:**

| Page | Line | Currently Reads | Should Read | Reason |
|------|------|-----------------|-------------|--------|
| | | | | |
| | | | | |

**Caution:** Substantive changes (changing "yes" to "no") are permissible but can be used for impeachment. Discuss with attorney before making substantive changes.

---

## Outputs from Post-Deposition Phase

1. **Transcript Review Summary** - Errors, irregularities, key passages
2. **Errata Sheet** (if corrections needed)
3. **Key Testimony Log** - Admissions, damaging statements with citations
4. **Objection Catalog** - All objections with waiver analysis
5. **Trial Preparation Memo** - Usable testimony, motion candidates, impeachment
6. **Case File Summary** - Comprehensive deposition summary

## Timeline

| Task | Deadline |

| Key testimony extraction | Within 2 weeks |
| Full analysis complete | Within 3 weeks |
| Case file update | Within 3 weeks |
