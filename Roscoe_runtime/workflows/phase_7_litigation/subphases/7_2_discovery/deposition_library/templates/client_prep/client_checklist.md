## Pre-Deposition Preparation (2-4 weeks before)

### Session 1: Initial Prep Meeting

- [ ] **Explain the Process**
  - [ ] What a deposition is and its purpose
  - [ ] Who will be present (opposing counsel, court reporter, videographer)
  - [ ] How long it typically takes
  - [ ] The binding nature of sworn testimony

- [ ] **Review Fundamental Rules**
  - [ ] Tell the truth
  - [ ] Listen to the entire question before answering
  - [ ] If you don't know, say "I don't know"
  - [ ] Don't guess or speculate
  - [ ] Answer only what is asked
  - [ ] Don't volunteer information
  - [ ] Take your time
  - [ ] Ask for clarification if needed

- [ ] **Review Key Documents**
  - [ ] Complaint and Answer
  - [ ] Client's prior statements (if any)
  - [ ] Medical records relevant to injuries claimed
  - [ ] Accident report
  - [ ] Photos of accident scene/injuries
  - [ ] Client's interrogatory answers

### Session 2: Topic Review

- [ ] **Review Expected Question Areas**
  - [ ] Background (name, address, family, employment)
  - [ ] Medical history prior to accident
  - [ ] Details of the accident
  - [ ] Injuries sustained
  - [ ] Medical treatment received
  - [ ] Current condition and limitations
  - [ ] Lost wages and impact on work
  - [ ] Impact on daily life
  - [ ] Prior accidents or injuries
  - [ ] Prior lawsuits or claims

- [ ] **Address Problem Areas**
  - [ ] Prior injuries to same body parts
  - [ ] Gaps in treatment
  - [ ] Social media content
  - [ ] Criminal history (if any)
  - [ ] Inconsistent statements
  - [ ] Unfavorable facts

- [ ] **Practice Key Topics**
  - [ ] Practice describing the accident clearly and concisely
  - [ ] Practice describing injuries without exaggeration
  - [ ] Practice handling difficult questions

### Session 3: Practice (if needed)

- [ ] **Mock Examination**
  - [ ] Conduct practice Q&A
  - [ ] Identify areas needing improvement
  - [ ] Reinforce good habits
  - [ ] Build client confidence

## Privilege Review

### Items to Identify and Protect

- [ ] Attorney-client communications reviewed
- [ ] Work product documents identified
- [ ] Any court-ordered limitations noted

### Topics for Instruction Not to Answer

Only instruct not to answer for:
1. [ ] Privileged attorney-client communications
2. [ ] Work product materials
3. [ ] Court-ordered prohibited topics

## Day Before Deposition

- [ ] Final review call/meeting with client
- [ ] Confirm logistics (time, location, parking)
- [ ] Remind client about:
  - [ ] What to wear
  - [ ] Arriving 30 minutes early
  - [ ] Getting good rest
  - [ ] Eating before (but not too much)
- [ ] Confirm court reporter and videographer (if applicable)

---

## Day of Deposition

### Before Deposition (30 min early meeting)

- [ ] Final rules review
- [ ] Address any last-minute concerns
- [ ] Quick review of key facts
- [ ] Ensure client is calm and ready

### During Deposition (Attorney Tasks)

- [ ] Monitor for objectionable questions
- [ ] Make timely objections (form only in discovery depo)
- [ ] Watch for:
  - [ ] Privilege incursions
  - [ ] Harassment or bad faith conduct
  - [ ] Client fatigue or confusion
- [ ] Request breaks as needed

### Monitor for Red Flags

| Red Flag | Response |
|----------|----------|
| Questions about attorney-client communications | Object; instruct not to answer |
| Repeated badgering | Object; warn of CR 30.04 motion |
| Personal attacks | Object; consider suspension |
| Client confusion | Request break to confer |
| Client fatigue | Request appropriate breaks |

---

## Post-Deposition

### Immediately After

- [ ] Debrief with client
- [ ] Note any concerns about testimony
- [ ] Discuss next steps

### When Transcript Arrives

- [ ] Review transcript for errors
- [ ] Identify key admissions (favorable and unfavorable)
- [ ] Note any testimony contradicting case theory
- [ ] Prepare errata sheet if needed (within time allowed)
- [ ] Update case file with deposition summary

## Form Objections Quick Reference

| Objection | When to Use |
|-----------|-------------|
| **Leading** | Question suggests the answer |
| **Calls for Speculation** | Asks witness to guess |
| **Asked and Answered** | Same question already answered |
| **Argumentative** | Harassing, not fact-seeking |
| **Assumes Facts Not in Evidence** | Presumes unestablished facts |
| **Calls for Legal Conclusion** | Asks for legal opinion |
| **Lacks Foundation** | No factual predicate established |
| **Compound** | Multiple questions in one |
| **Calls for Narrative** | Overly broad invitation to ramble |
