# Module: Cell Phone and Electronic Device Usage

## Purpose
These questions target cell phone and electronic device usage at the time of the collision to establish distracted driving. Critical when distraction is suspected.

## Usage
1. Copy questions to base template
2. Renumber sequentially
3. Customize time periods as needed

---
