# Module: Prior and Subsequent Incidents

## Purpose
These questions discover prior and subsequent similar incidents for notice, pattern evidence, and punitive damages support. Critical for premises and product liability cases.

## Usage
1. Copy questions to base template
2. Renumber sequentially
3. Customize description of "similar incident" for case type

---

## INTERROGATORIES - PRIOR AND SUBSEQUENT INCIDENTS

**INTERROGATORY NO. [X]:**
Identify all incidents prior to the subject incident that are substantially similar to Plaintiff's incident, including incidents that:
- (a) Occurred at the same location;
- (b) Occurred at a similar location on the premises;
- (c) Involved the same type of hazardous condition;
- (d) Resulted in similar injuries.

For each incident, state:
- (i) Date of incident;
- (ii) Location of incident;
- (iii) Name and contact information of person(s) involved;
- (iv) Description of what occurred;
- (v) Injuries or damages claimed;
- (vi) Outcome (claim, lawsuit, settlement).
