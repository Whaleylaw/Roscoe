# Deficiency Checklist Reference

## Interrogatory Response Issues

### Complete Non-Response

- [ ] No response provided
- [ ] Only objection, no answer
- [ ] "Will supplement" with no substance

**Action:** Demand complete response or seek motion to compel.

### Evasive Answers

- [ ] Answer doesn't address the question
- [ ] Generic response to specific question
- [ ] "To be determined" or "unknown at this time"
- [ ] Refers to documents without identification

**Action:** Point out specific deficiency, request direct answer.

### Incomplete Answers

- [ ] Partial list ("including but not limited to")
- [ ] Missing requested details
- [ ] Answer qualified without explanation

**Action:** Request complete response with all information.

### Improper Objections

- [ ] Objection without grounds stated
- [ ] Boilerplate objections
- [ ] Objection only, no answer subject to objection
- [ ] Objection to standard request

**Action:** Challenge validity, demand answer subject to objection.

---

## RFP Response Issues

### No Production

- [ ] No documents produced
- [ ] Documents promised but not provided
- [ ] Objection only, no search conducted

**Action:** Demand production or certification that none exist.

### Incomplete Production

- [ ] Some documents but obviously incomplete
- [ ] Production doesn't match response description
- [ ] Missing categories of documents

**Action:** Identify gaps, demand supplementation.

### Privilege Issues

- [ ] Privilege claimed without log
- [ ] Privilege log incomplete
- [ ] Overbroad privilege claims

**Action:** Demand proper privilege log, challenge if overbroad.

### Production Format Issues

- [ ] Documents disorganized
- [ ] No Bates numbering
- [ ] Illegible copies
- [ ] Metadata stripped (if electronic)

**Action:** Request organized re-production.

---

## RFA Response Issues

### Improper Denials

- [ ] Denial of clearly true fact
- [ ] Denial without explanation
- [ ] "Unable to admit or deny" without inquiry

**Action:** Note for cost recovery at trial (CR 37(c)).

### Qualified Responses

- [ ] Admits "in part" without specificity
- [ ] "Subject to" qualifications that negate admission

**Action:** Demand unqualified response.

### Non-Response

- [ ] No response (deemed admitted after 30 days)
- [ ] Late response without motion

**Action:** Consider motion to deem admitted.

## Deficiency Letter Template

```
RE: Deficient Discovery Responses - [Case Name]

Dear Counsel:

We have reviewed Defendant's responses to Plaintiff's [First Set 
of Interrogatories/Requests for Production] served on [DATE]. We 
write to identify the following deficiencies pursuant to our 
obligation to meet and confer before filing any motion to compel.

DEFICIENCIES:

Interrogatory No. [X]: [Description of deficiency]

Request for Production No. [X]: [Description of deficiency]

We request that you provide supplemental responses addressing 
these deficiencies within [14] days. If we cannot resolve these 
issues informally, we will be compelled to seek court intervention.

Please contact us to schedule a meet and confer conference.
```
