# Exhibit Management Reference

## Pre-Deposition Preparation

### Exhibit Selection

Choose exhibits that:
- Support your case theory
- May trigger admissions
- Need authentication
- Will impeach witness

### Organization

| Exhibit # | Description | Purpose | Page |
|-----------|-------------|---------|------|
| 1 | Accident report | Authenticate, admissions | 15 |
| 2 | Scene photos | Identify, describe | 20 |
| 3 | Vehicle photos | Damage, identify | 25 |
| 4 | Discovery response | Pin down, impeach | 30 |

### Copies Needed

Prepare copies for:
- [ ] Witness
- [ ] Opposing counsel
- [ ] Court reporter
- [ ] Your file

## During Deposition

### Marking Exhibits

```
Q: I'm going to hand you what I've had marked as Deposition 
   Exhibit 1. Do you recognize this document?

A: [Response]

Q: What is it?

A: [Response]
```

### Authentication

For documents:
```
Q: Is this a true and accurate copy of [document]?
Q: Were you the author of this document?
Q: Is this your signature?
```

For photos:
```
Q: Does this photograph fairly and accurately depict [subject] 
   as it appeared on [date]?
```

### Using Exhibits for Impeachment

```
Q: You just testified that [X]. Is that correct?
A: Yes.
Q: Please look at Exhibit 3. Can you read the highlighted portion?
A: [Reads]
Q: Does that refresh your recollection?
```

## Post-Deposition

### Exhibit Handling

- [ ] Ensure all exhibits attached to transcript
- [ ] Note which exhibits admitted vs. objected
- [ ] Organize exhibits with case file
- [ ] Index exhibits for trial use

### Transcript Review

When reviewing transcript:
- Flag exhibit references
- Note authentication obtained
- Identify impeachment opportunities
- Track admissions made re: exhibits
