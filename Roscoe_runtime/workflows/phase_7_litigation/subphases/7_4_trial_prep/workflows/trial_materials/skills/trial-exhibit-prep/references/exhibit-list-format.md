# Exhibit List Format Reference

## Standard Format

```
PLAINTIFF'S EXHIBIT LIST

[Case Caption]
Civil Action No. [Number]

| Ex. # | Description | Pages/Bates | Foundation Witness | Objection Status |
|-------|-------------|-------------|--------------------|--------------------|
| P-1 | [Description] | [Range] | [Witness] | [Stipulated/Contested] |
| P-2 | [Description] | [Range] | [Witness] | [Stipulated/Contested] |

TOTAL EXHIBITS: [X]

                    Respectfully submitted,
                    [Signature block]
```

## Common PI Exhibits

### Liability Exhibits

| Type | Foundation |
|------|------------|
| Police report | Officer or business record |
| Scene photos | Photographer or witness |
| Vehicle photos | Photographer or party |
| Surveillance video | Custodian of records |
| Accident diagram | Expert or reconstructionist |

### Medical Exhibits

| Type | Foundation |
|------|------------|
| Medical records | Custodian or treating physician |
| Medical bills | Custodian or billing dept |
| X-rays/MRIs | Radiologist or treating physician |
| Photos of injuries | Photographer or plaintiff |

### Damage Exhibits

| Type | Foundation |
|------|------------|
| Pay stubs | Employer or plaintiff |
| Tax returns | Plaintiff |
| Life care plan | Life care planner |
| Economic report | Economist |

### Demonstrative Exhibits

| Type | Foundation |
|------|------------|
| Anatomical models | Expert |
| Animations | Expert who created/verified |
| Timelines | Attorney or witness |
| Summary charts | FRE 1006 foundation |

## Organization Tips

### Numbering System

```
P-1 through P-[X]   Plaintiff exhibits
D-1 through D-[X]   Defendant exhibits
J-1 through J-[X]   Joint exhibits
```

### Ordering Strategy

Option 1: **Chronological**
- Exhibits in order of events
- Natural storytelling flow

Option 2: **By Witness**
- Group by foundation witness
- Efficient presentation

Option 3: **By Topic**
- Liability exhibits together
- Damages exhibits together

---

## Stipulations

Try to stipulate to:
- Medical records authenticity
- Business records authenticity
- Photo accuracy
- Basic documents

Creates cleaner trial, saves time.

---

## Objection Handling

Common objections to anticipate:
- Hearsay
- Relevance
- Foundation
- Authentication
- Best evidence
- Prejudicial

Prepare responses in advance.
