# Trial Materials Workflow

## Overview

Prepare and file all required pretrial materials to ensure readiness for trial.

## Entry Criteria

- Trial date set
- Pretrial deadlines identified
- Discovery complete

## Steps

### 1. Prepare Exhibit List

**Owner:** Agent  
**Skill:** `trial-exhibit-prep`  
**Action:** Compile list of all exhibits with Bates numbers.

### 2. Prepare Witness List

**Owner:** Agent  
**Action:** List all witnesses with contact info and summary of testimony.

### 3. Draft Pretrial Brief

**Owner:** Agent  
**Action:** Create pretrial memorandum with case summary, legal issues.

### 4. Prepare Jury Instructions

**Owner:** Agent  
**Action:** Draft proposed jury instructions.

### 5. Organize Trial Notebook

**Owner:** Agent  
**Action:** Compile all materials in trial organization system.

### 6. File Required Documents

**Owner:** User  
**Action:** File all pretrial materials by deadlines.

### 7. Attend Pretrial Conference

**Owner:** User (Attorney)  
**Action:** Attend pretrial conference, resolve disputes.

## Exit Criteria

- [ ] Exhibit list filed
- [ ] Witness list filed
- [ ] Pretrial brief filed
- [ ] Jury instructions submitted
- [ ] Trial notebook organized
