# Sub-Phase 7.4: Trial Prep

## Overview

**Sub-Phase ID:** `7.4_trial_prep`  
**Parent Phase:** 7 (Litigation)  
**Track:** Litigation

Trial Prep is the final preparation before trial. Includes managing expert witnesses, preparing trial exhibits, filing pretrial briefs, and ensuring all materials are ready for trial.

---

## Entry Triggers

- Mediation unsuccessful
- Case set for trial
- Pretrial deadlines approaching

---

## Exit Criteria

| Criterion | Description | Verification |
|-----------|-------------|--------------|
| Trial Ready | All pretrial requirements complete | Per court scheduling order |

---

## Workflows

| Workflow | Purpose | Path |
|----------|---------|------|
| expert_management | Manage expert disclosures and preparation | `workflows/expert_management/workflow.md` |
| trial_materials | Prepare exhibits, witness list, pretrial filings | `workflows/trial_materials/workflow.md` |

---

## Key Deadlines

| Deadline | Typical Timing | Item |
|----------|----------------|------|
| Expert disclosure | 90 days before trial | Our experts |
| Rebuttal expert | 60 days before trial | Response to their experts |
| Exhibit list | 14-30 days before trial | Per scheduling order |
| Witness list | 14-30 days before trial | Per scheduling order |
| Pretrial brief | 7-14 days before trial | Per local rules |
| Jury instructions | Per court | Proposed instructions |
