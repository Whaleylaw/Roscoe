# Trial Presentation Skill

## Overview

Present plaintiff's case effectively through all phases of trial.

## When to Use

Use when:
- Delivering opening statement
- Conducting direct examination
- Cross-examining defense witnesses
- Delivering closing argument

DO NOT use if:
- Preparing for trial (use trial-exhibit-prep)
- Pre-trial motions (different skill)

## Workflow

### Step 1: Opening Statement

Tell the jury what to expect:
- Theme of case
- Key facts
- Damages overview
- What you'll prove

**See:** `references/opening-structure.md` for format.

### Step 2: Direct Examination

Present evidence through witnesses:
- Open-ended questions
- Build story chronologically
- Introduce exhibits
- Establish elements

**See:** `references/direct-examination.md` for technique.

### Step 3: Closing Argument

Tie everything together:
- Review evidence
- Apply to jury instructions
- Address weaknesses
- Ask for specific damages

**See:** `references/closing-structure.md` for format.

## Output Format

```markdown

### Opening

2. [Witness] - [Purpose]

- [Key argument 2]
- Damages request: $[Amount]
```
