## Template Variables Reference

| Variable | Description | Example |
|----------|-------------|---------|
| `{client_first_name}` | Client's first name | Jane |
| `{client_full_name}` | Client's full name | Jane Smith |
| `{case_name}` | Case folder name | Jane-Smith-MVA-12-05-2024 |
| `{case_type}` | Case type code | MVA |
| `{sender_name}` | Staff member name | John at Whaley Law |
| `{required_documents_list}` | Bulleted list of required docs | See below |
| `{additional_documents_list}` | Bulleted list of optional docs | See below |
| `{missing_documents_list}` | Bulleted list of still-needed docs | See below |
| `{document_name}` | Single document name | Medical Authorization |
| `{document_description}` | Description of document purpose | See below |

## Document List Formatting

### Required Documents List Example

```
• New Client Information Sheet - Basic information about you and your case
• Fee Agreement - Our representation agreement (please sign and return)
• Medical Authorization (HIPAA) - Allows us to obtain your medical records
```

### Additional Documents List Example

```
• Medical Treatment Questionnaire - Details about your injuries and treatment
• Digital Signature Authorization - Allows us to use your signature electronically
• Accident Detail Information Sheet - Specific details about how the accident occurred
```

### Document Descriptions

| Document | Description |
|----------|-------------|
| New Client Information Sheet | "This form collects your basic contact and personal information." |
| Fee Agreement | "This is our legal representation agreement outlining our services and fee structure. Please review, sign, and return." |
| Medical Authorization (HIPAA) | "This authorization allows us to request your medical records from healthcare providers. Without this, we cannot obtain records on your behalf." |
| Medical Treatment Questionnaire | "This helps us understand your injuries and the treatment you've received." |
| Digital Signature Authorization | "This allows us to use your electronic signature on documents, making the process faster and more convenient." |
| Accident Detail Sheet | "This form captures specific details about how your accident occurred, which is essential for building your case." |
| Wage & Salary Verification | "If you've missed work due to your injury, this helps us document your lost wages." |
| CMS Medicare Verification | "If you are on or approaching Medicare, this form is required for federal compliance." |
