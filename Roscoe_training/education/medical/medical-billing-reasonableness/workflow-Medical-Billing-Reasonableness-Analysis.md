# Medical Billing Reasonableness Analysis

## 1. Operational Workflow

### Workflow Name
**Medical Billing Reasonableness Analysis Workflow**

---

### Goal
Produce a legally defensible opinion on the reasonable value of medical expenses that:
- Meets the "reasonable and necessary" evidentiary standard required in 46+ jurisdictions
- Can withstand Daubert challenges and collateral source objections
- Provides data-driven benchmarks against billed charges
- Identifies billing errors, coding irregularities, and statistical outliers
- Supports demand letter calculations, settlement negotiations, and trial preparation

---

### When to Use
- **Preparing damages calculations** for demand letters or trial exhibits
- **Reviewing medical bills** for coding compliance before submission
- **Coordinating with billing experts** on opinion testimony
- **Settlement negotiations** when defense raises medical specials reduction arguments
- **Responding to defense challenges** on billing reasonableness
- **Pre-litigation case evaluation** to assess damages exposure
- **Any time billed charges exceed typical market rates** or defense contests medical specials

---

### Inputs Required

| Input | Description | Source |
|-------|-------------|--------|
| Complete medical bills | Itemized bills with CPT/HCPCS codes, dates, charges | Provider billing departments |
| Medical records | Clinical documentation supporting services rendered | Treatment providers |
| Provider information | Name, NPI number, facility type, ZIP code | Bills or NPI registry |
| CPT/HCPCS codes | All procedure codes billed | Itemized bills |
| Diagnosis codes (ICD-10) | Diagnoses supporting medical necessity | Medical records |
| DRG codes | For inpatient hospital stays | Hospital bills |
| NDC codes | For pharmaceutical charges | Pharmacy records |
| Dates of service | When each service was rendered | Bills and records |
| Case jurisdiction | State/federal court location | Case file |
| Treatment timeline | Chronology of care | Medical chronology |

---

### Step-by-Step Process

#### **Phase 1: Document Collection & Organization**

**Step 1.1: Gather Complete Billing Records**
- Collect all itemized medical bills (not summary statements)
- Ensure bills contain line-item CPT/HCPCS codes
- Verify dates of service are clearly indicated
- Obtain provider NPI numbers and addresses
- Request any missing billing codes from providers

**Step 1.2: Compile Supporting Medical Records**
- Match clinical documentation to each billed service
- Ensure records support medical necessity for each procedure
- Identify the treating provider for each service
- Note any referrals or authorizations documented

**Step 1.3: Categorize Services by Type**
- Professional services (physician/practitioner)
- Inpatient facility services
- Outpatient facility services
- Anesthesia services
- Pharmaceutical services
- Durable Medical Equipment (DMEPOS)

---

#### **Phase 2: Technical Coding Audit (Prong 1)**

**Step 2.1: Verify Code Accuracy**
- Confirm CPT/HCPCS codes match documented procedures
- Cross-reference diagnosis codes (ICD-10) with procedures
- Verify codes are current for date of service
- Check that code descriptions match clinical notes

**Step 2.2: Bundling/Unbundling Analysis**
- Cross-reference code combinations against NCCI edits
- Identify any separately billed codes that should be bundled
- Flag potential unbundling violations
- Document any legitimate modifier exceptions

**Step 2.3: Modifier Review**
- Verify all modifiers are appropriately applied
- Confirm bilateral procedure modifiers (-50, -RT, -LT)
- Check multiple procedure modifiers (-51, -59)
- Validate E&M service modifiers (-25)

**Step 2.4: E&M Level Appropriateness**
- Compare E&M code level to documentation complexity
- Apply 1995 or 1997 documentation guidelines
- Flag potential upcoding where documentation is thin
- Ensure time-based billing has documented time

**Step 2.5: Service-Specific Audits**

*For Professional Services:*
- Apply AMA CPT coding rules
- Check CCI edits for bundling
- Verify modifier usage

*For Anesthesia Services:*
- Calculate: Base units + Time units + Modifying units
- Verify ASA code assignment
- Confirm physical status modifier
- Validate qualifying circumstances

*For Inpatient Facility:*
- Verify DRG assignment accuracy
- Check length of stay appropriateness
- Review discharge status
- Apply wage index adjustments
- Cross-reference American Hospital Directory

*For Outpatient Facility:*
- Verify APC code assignments
- Check status indicators
- Review CPT-to-APC mapping

*For Pharmaceuticals:*
- Identify NDC codes
- Cross-reference AWP from RED BOOK
- Check for appropriate dosing
- Verify administration codes

*For DME:*
- Verify HCPCS codes
- Compare to state fee schedules
- Check certificate of medical necessity

---

#### **Phase 3: Geographic Pricing Analysis (Prong 2)**

**Step 3.1: Prepare Pricing Query Parameters**
- Compile validated CPT/HCPCS codes from Prong 1
- Record exact dates of service
- Obtain provider ZIP codes
- Note facility type (hospital, ASC, office)

**Step 3.2: Query National Databases**
- Access UCR databases (Fair Health, Optum, PMIC, Context4Healthcare)
- Input CPT code + date + ZIP code
- Retrieve percentile benchmarks:
  - 50th percentile (median market rate)
  - 75th percentile (above-average market rate)
  - 90th percentile (high-end market rate)

**Step 3.3: Calculate Variance from Market**
- Compare billed charge to each percentile
- Calculate percentage above/below benchmarks
- Flag charges exceeding 90th percentile as outliers
- Document variance for each line item

**Step 3.4: Government Fee Schedule Comparison**
- Obtain Medicare Physician Fee Schedule (MPFS) rates
- Calculate commercial multipliers (typically 1.4x–2.5x Medicare)
- Compare billed charges to Medicare-derived ranges
- Note any charges exceeding commercial range ceiling

---

#### **Phase 4: Local Market Validation (Prong 3)**

**Step 4.1: Hospital Transparency Data**
- Access hospital chargemaster/transparency files
- Review negotiated rates with major insurers
- Compare to billed charges
- Document range of accepted payments

**Step 4.2: State Healthcare Cost Databases**
- Query state All-Payer Claims Database (if available)
- Review state-specific fee schedules
- Check workers' compensation fee schedules
- Note any state-mandated rate caps

**Step 4.3: Medicare Fee Schedule Validation**
- Cross-reference with CMS MPFS lookup
- Check OPPS rates for outpatient facility
- Review IPPS rates for inpatient facility
- Document Medicare baseline for each service

**Step 4.4: Professional Experience Overlay**
- Note any known local market factors
- Consider facility reputation/specialization
- Account for legitimate cost differentials
- Document rationale for any adjustments

---

#### **Phase 5: Reasonable Value Determination**

**Step 5.1: Select Appropriate Benchmark**
- Standard services: Use 75th percentile as reasonable value
- Complex/specialized services: Consider higher percentile
- Documented outliers: Use 90th percentile as ceiling
- Document rationale for percentile selection

**Step 5.2: Calculate Reasonable Value Summary**
- Create line-item comparison table:
  - Service/CPT code
  - Billed amount
  - 50th/75th/90th percentile
  - Recommended reasonable value
  - Variance from billed

**Step 5.3: Calculate Total Reasonable Medical Specials**
- Sum all individual reasonable values
- Calculate total reduction from billed charges
- Express as percentage reduction
- Prepare summary table

---

#### **Phase 6: Documentation & Reporting**

**Step 6.1: Prepare Analysis Report**
- Executive summary with key findings
- Methodology description (three-pronged approach)
- Detailed line-item analysis tables
- Red flags and billing issues identified
- Reasonable value conclusions

**Step 6.2: Prepare Evidentiary Support**
- Document data sources used
- Cite database query parameters
- Preserve percentile lookup results
- Prepare Daubert-ready methodology summary

**Step 6.3: Identify Expert Coordination Needs**
- Flag complex issues requiring expert consultation
- Note areas where testimony may be challenged
- Prepare questions for billing expert
- Compile supporting documentation for expert

---

### Quality Checks & Safeguards

#### **Coding Accuracy Checks**
- [ ] All CPT codes verified against medical record documentation
- [ ] NCCI bundling edits applied and documented
- [ ] Modifiers reviewed for appropriate usage
- [ ] E&M levels supported by documentation complexity
- [ ] DRG assignments verified for inpatient stays

#### **Data Integrity Checks**
- [ ] ZIP codes verified for all providers
- [ ] Dates of service confirmed accurate
- [ ] NPI numbers validated
- [ ] All codes current for service dates
- [ ] Database queries documented and reproducible

#### **Red Flags Requiring Attorney Escalation**
| Red Flag | Description | Escalation Action |
|----------|-------------|-------------------|
| **Charges >90th Percentile** | Billed amount significantly exceeds market ceiling | Flag for expert review; potential damages reduction |
| **Unbundling Patterns** | Multiple separate codes should be bundled | Potential billing fraud; expert analysis needed |
| **Upcoding Indicators** | High-level codes unsupported by documentation | Verify with records; may need provider inquiry |
| **Services Not Documented** | Billed services lack clinical documentation | Challenge necessity; request supporting records |
| **Duplicate Billing** | Same service billed multiple times | Document for reduction; potential fraud indicator |
| **Daubert Challenge Signals** | Defense attacking methodology/sources | Prepare expert defense; gather supporting authority |
| **Collateral Source Objection** | Defense claims improper insurance evidence | Deploy Cameron v. Lowes rebuttal |
| **Lopez Argument** | Defense cites Lopez re: stipulated reasonableness | Distinguish—reasonableness not stipulated here |

#### **Legal Compliance Checks**
- [ ] Analysis based on market data, not specific insurance payments
- [ ] Methodology consistent with FRE 702/703 requirements
- [ ] Data sources are industry-standard and peer-accepted
- [ ] Analysis reproducible by other qualified experts
- [ ] No legal conclusions—framed as support for attorney
