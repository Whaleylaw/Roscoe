# ElevenLabs Text-to-Speech Audio Files

All audio files generated by the ElevenLabs text-to-speech tool are saved to the `audio_outputs/` directory.

## Quick Access

### Play the Most Recent Audio
```bash
./play_latest.sh
```

This script will automatically find and play the most recently generated audio file.

### Browse All Audio Files
```bash
open audio_outputs/
```

Or list them:
```bash
ls -lht audio_outputs/*.mp3
```

### Play a Specific File
```bash
afplay audio_outputs/tts_20231116_202500.mp3
```

## File Naming

Audio files are named with timestamps:
- Format: `tts_YYYYMMDD_HHMMSS.mp3`
- Example: `tts_20231116_202500.mp3` = Generated on Nov 16, 2023 at 8:25:00 PM

## Details

- **Format**: MP3 (44.1kHz, 128kbps, Mono)
- **Voice**: ElevenLabs voice ID `NNl6r8mD7vthiJatiJt1`
- **Model**: `eleven_multilingual_v2`
- **Location**: `./audio_outputs/`

## Management

The `audio_outputs/` directory is excluded from git (in `.gitignore`), so audio files won't be committed to the repository.

To clean up old audio files:
```bash
rm audio_outputs/*.mp3
```
